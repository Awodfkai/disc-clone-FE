# Read Me Discord Project

## Features

* Servers
  * Create
  * Join
  * Roles
    * different permissions on server
  * Channels
    * create
    * delete
    * join
    * different permissions for different roles
* Groups
  * Create
  * Add other users
  * Auto-created between individual users when sending a DM
* Live text chat
  * in groups
  * in channels

## Technologies

* Socket.IO for live text chat
